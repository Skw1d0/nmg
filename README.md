# Nmg

## Was ist Nmg

Nmg ist eine private Webanwendung zum Erheben von Schutzmaßnahmen im operativen Notfallmanagement der DB InfraGO AG.

## Was bietet Nmg?

Die App bietet:

- Erstellen von Schutzmaßnahmen im Notfallmanagement.
- Speichern von Schutzmaßnahmen im Notfallmanagement.
- Exportieren der Schutzmaßnahme in den Vordruck 123.2110V01 der Deutschen Bahn AG nach Eingabe eines Kennworts. Das
  Dokument 123.2110V01 ist verschlüsselt abgelegt und kann nur mit dem richtigen Kennwort entschlüsselt werden.

## Wo werden die Daten gespeichert?

Die Daten werden im Store des Browsers gespeichert. Sie sind somit nur auf dem Gerät gespeichert, auf dem man die App
verwendet.
